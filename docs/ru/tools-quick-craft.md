# Инструменты — Валюта — Quick Craft

## Что делает
Показывает оверлеи валюты около курсора (глобальная горячая клавиша) или по элементам. Один клик ЛКМ по оверлею выполняет быстрое действие: ПКМ по источнику и ЛКМ обратно по курсору — без смены фокуса.

## Настройка
1. Инструменты → Валюта: добавьте записи (изображение, зона захвата, активность).
2. Инструменты → Quick Craft: задайте глобальную горячую клавишу, разместите окна (центр‑гайд), сохраните.
3. Добавьте EXE игры в `assets/allowed_processes.json` для режима «только при фокусе игры».

## Использование
- Нажмите глобальную горячую клавишу → оверлеи появятся около курсора.
- Кликните по оверлею один раз → действие выполнится сразу.

## Примечания
- Оверлеи не появляются на панели задач/Alt‑Tab и не забирают фокус.
- Работает при фокусе игры (или когда активно приложение, если это разрешено в настройках).
