# Двойной Ctrl

[English](../en/double-ctrl-click.md) | **Русский**

## Обзор
Двойное нажатие Ctrl запускает непрерывную эмуляцию ЛКМ; отпускание Ctrl её прекращает. Обнаружение фронта исключает повторный запуск из-за автоповтора.

## Настройка
1. В **Инструменты → Mega QoL** включите **Double Ctrl Click** по необходимости.
2. Убедитесь, что политика фокуса и список разрешённых процессов настроены как нужно.

## Примечания
- Работает только при фокусе игры (или при фокусе приложения PathOfQuality, если снято ограничение по фокусу).
- Индикатор на плавающей панели показывает состояние эмуляции кликов.
