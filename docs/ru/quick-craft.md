# Quick Craft

## Обзор
Вызывает оверлеи валюты рядом с курсором и выполняет последовательность: ПКМ по источнику → ЛКМ обратно по курсору, — ни разу не перехватывая фокус окна.

## Настройка
1. В **Инструменты → Валюта** добавьте записи с изображением, зоной захвата и состоянием активности.
2. В **Инструменты → Quick Craft** задайте глобальную горячую клавишу, расположите шаблоны оверлеев с помощью центрального ориентира и сохраните.
3. (Опционально) Добавьте исполняемый файл игры в `assets/allowed_processes.json`, чтобы ограничить выполнение фокусом игры.

## Использование
- Нажмите глобальную горячую клавишу — оверлеи появятся рядом с курсором.
- Одна ЛКМ по оверлею моментально запускает быстрое действие.

## Советы
- Оверлеи невидимы для Alt+Tab и панели задач и не активируют окна.
- Работает при фокусе игры; при отключении ограничения также действует, когда активно приложение PathOfQuality.
