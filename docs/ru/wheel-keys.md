# Wheel Keys

[English](../en/wheel-keys.md) | **Русский**

## Обзор
При прокрутке колеса вниз отправляет настраиваемую последовательность клавиш (1–4) с задержкой. Подавление «бурстов» запускает только одну последовательность на серию прокрутки.

## Настройка
1. Откройте **Инструменты → Mega QoL** и включите **Wheel → Keys**.
2. Задайте последовательность клавиш (например, `1,2,3,4`) и задержку между нажатиями в миллисекундах.

## Примечания
- Учитывает политику фокуса и список разрешённых процессов.
- Защищает от спама даже при быстрой прокрутке.
