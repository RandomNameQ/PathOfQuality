# Библиотека — Копия области экрана

## Что делает
Показывает живую вырезку экрана как оверлей. Режимы:
- Всегда показывать, или
- По условию: показывать, когда выбранного баффа/дебаффа НЕТ на экране.

## Как использовать
1. Добавьте область в Библиотека → Области копирования (прямоугольник захвата, размер, позиция, прозрачность).
2. При необходимости свяжите ссылки на баффы/дебаффы для условного показа.
3. Переключайте «Копию» в Обзоре или на плавающей панели.

## Примечания
- При наведении оверлей временно скрывается — удобно для взаимодействия с UI.
- Позиционирование/масштабирование: колесо делает размер квадрата.
