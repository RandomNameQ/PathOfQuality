# Инструменты — Mega QoL — Двойной Ctrl

## Что делает
Двойное нажатие Ctrl запускает непрерывную эмуляцию ЛКМ. При отпускании Ctrl эмуляция останавливается. Используется обнаружение «фронта нажатия», чтобы избежать ложных срабатываний из‑за автоповтора.

## Настройка
1. Инструменты → Mega QoL: включите Double Ctrl Click при необходимости.
2. Убедитесь, что политика фокуса и «разрешённые процессы» настроены под ваши предпочтения.

## Примечания
- Работает только при фокусе игры (или при фокусе приложения, если режим разрешён).
- Индикатор на плавающей панели показывает состояние эмуляции кликов.
