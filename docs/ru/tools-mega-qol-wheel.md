# Инструменты — Mega QoL — Wheel Keys

## Что делает
При прокрутке колеса вниз отправляет настраиваемую последовательность клавиш (1–4) с задержкой. Подавление «бурстов» гарантирует один запуск на серию прокрутки.

## Настройка
1. Инструменты → Mega QoL: включите Wheel→Keys.
2. Задайте последовательность (например, `1,2,3,4`) и задержку (мс).

## Примечания
- Учитывает политику фокуса и список разрешённых процессов.
- Избегает спама при быстрой прокрутке.
