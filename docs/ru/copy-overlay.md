# Оверлей копирования

[English](../en/copy-overlay.md) | **Русский**

## Обзор
Показывает «живую» вырезку экрана в отдельном оверлее. Можно держать его всегда видимым или включать только при отсутствии связанных баффов/дебаффов.

## Настройка
1. Перейдите в **Библиотека → Области копирования** и добавьте новую запись (прямоугольник захвата, размер, позиция, прозрачность).
2. При необходимости привяжите область к отслеживаемым баффам/дебаффам, чтобы управлять условием показа.

## Использование
- Включайте оверлей из **Обзора** или с плавающей панели.
- Когда условие выполняется, захваченная область обновляется в реальном времени внутри окна оверлея.

## Советы
- Наведение курсора временно скрывает оверлей для взаимодействия с интерфейсом под ним.
- В режиме позиционирования перетаскивайте и меняйте размер колесом мыши — пропорции сохраняются квадратными.
