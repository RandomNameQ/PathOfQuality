# Библиотека — Бафф/Дебафф Оверлей

## Что делает
Находит сохранённые иконки баффов/дебаффов на экране (внутри ROI) и рисует лёгкие «неактивирующиеся» оверлеи.

## Как использовать
1. Добавьте элементы в Библиотека → Баффы / Дебаффы (изображение, размер, позиция, прозрачность).
2. Настройте ROI в Настройках (Выбрать область).
3. Запустите Сканирование в Обзоре (или с плавающей панели).

## Примечания
- Оверлеи скрываются при наведении, чтобы можно было кликать по интерфейсу под ними.
- В режиме позиционирования можно перетаскивать/масштабировать; колесо делает квадратный размер.
